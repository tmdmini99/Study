##  럼바우의 객체지향 분석 기법

- 객체 모델링 : 객체 다이어그램으로 표시, 가장 중요시 선행
	- 객체 다이어그램(Object Diagram)
- 동적 모델링 : 상태 다이어그램(상태도), 동적인 흐름 행위
	- 상태 다이어그램(State Diagram)
- 기능 모델링 : 자료의 흐름을 이용하여 프로세스간의 자료 흐름을 처리
	- 자료 흐름도(Data Flow Diagram)
