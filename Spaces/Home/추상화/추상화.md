# [[추상클래스]]   
 추상 클래스는 인스턴스, 즉 객체를 만들 수 없다. 즉, new를 사용할 수 없다.
Animal animal = new Animal(); // ERROR, 'Animal' is abstract; cannot be instantiated 불가능
추상 메서드는 하위 클래스에게 메서드의 구현을 강제한다. (오버 라이딩이 강제된다.)
추상 메서드를 포함하는 클래스는 반드시 추상 클래스여야 한다.
